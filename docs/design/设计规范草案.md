# 设计规范草案

## 基础参考

ForAI 是核心视觉基础。需要优先还原：

- 黑白极简
- 高级留白
- 大字号标题
- 强网格系统
- 导航方式
- 项目卡片比例
- 边框和分割线
- 作品墙组织方式

## ControlRookie 差异化

在 ForAI 基础上加入工业自动化语义：

- PLC / CODESYS / MQTT / SmartControl
- 状态机、协议链、验证门禁
- 工程资产、工具、技能包、文章专题

## 动效原则

- 优先静态视觉完整度。
- 只使用低维护动效。
- 可接受 hover、fade、简单 marquee。
- 不强制复刻复杂滚动和页面切换动效。
