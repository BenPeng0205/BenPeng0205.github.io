# 设计规范草案

## 基础参考

ForAI 是核心视觉基础。后续 ControlRookie 官网必须优先还原：

- 黑白极简配色。
- 大面积留白。
- 轻字重、大字号、居中首屏标题。
- 顶部固定导航：左侧品牌黑点和名称，中间导航，右侧 Contact 胶囊按钮。
- 灰色说明文案，重点使用 `#565656` 一类低饱和灰。
- 首屏主视觉使用浅灰玻璃/液态球体，不使用复杂工业插画。
- 作品墙使用黑白灰卡片网格，少量 hover 即可。

## ControlRookie 差异化

在 ForAI 基础上加入工业自动化语义，但不能破坏极简审美：

- 内容关键词：PLC、CODESYS、MQTT、SmartControl、LicOS、AI 自动化、工程效率。
- 页面表达：工程资产、工具、技能包、文章专题、产品化服务。
- 视觉处理：只在局部卡片、文案、作品命名中体现工业语义，不在首屏堆叠流程图、仪表盘或复杂科技装饰。

## 动效原则

- 优先静态视觉完整度。
- 只使用低维护动效：hover、fade、轻量 marquee。
- 不强制复刻 ForAI 的复杂滚动、页面切换或 Framer 动画状态。
- 主视觉默认静态可见，避免动画初态导致截图或加载时空白。

## 验收门禁

- 桌面端截图：`1440x1400`。
- 移动端截图：`390x1200`。
- 检查项：首屏主视觉可见、标题不裁切、导航不溢出、移动端无横向滚动、中文文档无乱码。
