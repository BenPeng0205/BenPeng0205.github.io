# Open Design 首页高保真原型

本目录是当前唯一保留的 ControlRookie 首页高保真原型。

## 文件说明

- `index.html`：Open Design 生成并经校准后的首页原型。
- `screenshot-desktop.png`：桌面端验收截图，视口 `1440x1400`。

- FrontendDesign 不再作为首页主路线，只在需要探索内容密集区时辅助使用。
