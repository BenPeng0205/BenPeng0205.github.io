# Open Design 首页高保真原型

本目录是当前唯一保留的 ControlRookie 首页高保真原型。

## 文件说明

- `index.html`：Open Design 生成并经校准后的首页原型。
- `screenshot-desktop.png`：桌面辅助截图。
- `screenshot-mobile.png`：历史移动端参考截图，不作为当前高保真首页主验收依据。

## 使用原则

- 后续首页设计迭代直接修改本目录产物。
- ForAI 原站 HTML 不再作为开发底座，只作为视觉理解来源。
- FrontendDesign 不再作为首页主路线，只在需要探索内容密集区时辅助使用。
- 当前高保真验收必须用桌面浏览器实际打开页面，优先 Edge；移动端适配放到正式响应式阶段单独验证。
